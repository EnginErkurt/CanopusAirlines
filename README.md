Canopus Airlines - Airline Reservation System

Canopus Airlines is a full-stack airline reservation system developed as a web application using **ASP.NET, HTML, CSS, JavaScript, and SQL Server**. The project simulates the core functionality of a real airline website, allowing users to search for flights, book tickets, manage reservations, and providing administrators with a complete management panel.

Features

### User Features
- User registration and login
- Browse available flights
- Search flights by destination and date
- Book flight tickets
- View reservation history
- Cancel reservations
- Responsive and user-friendly interface

Admin Features
- Secure admin login
- Add new flights
- Edit flight information
- Delete flights
- View all reservations
- Manage flight schedules
- Monitor booking activities

Technologies Used

Frontend
- HTML5
- CSS3
- JavaScript

Backend
- ASP.NET
- C#

Database
- SQL Server

System Overview

The application follows a classic web-based client-server architecture.

Users can:

- Create an account and log in
- View available flights
- Reserve airline tickets
- Access their reservation history
- Manage existing bookings

Administrators can:

- Create, update, and delete flights
- View all customer reservations
- Manage flight information through an admin dashboard
